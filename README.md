# Fragrantica Redesign

Redesain website Fragrantica sebagai platform discovery parfum yang lebih modern, 
clean, dan ramah pengguna khususnya Gen Z.

## Preview
https://fathas1414.github.io/fragrantica-redesign/

## Deskripsi
Website ini merupakan hasil redesain homepage Fragrantica yang dikerjakan sebagai 
Final Project mata kuliah Responsive Web Design. Dibangun menggunakan HTML5, CSS3, 
dan JavaScript vanilla dengan tampilan yang responsif untuk Desktop, Tablet, dan Mobile.

## Fitur
- Hero Section dengan Fragrance Finder CTA
- Trending This Week dengan filter kategori
- Explore by Scent Notes
- Explore by Mood
- Find Your Perfect Fragrance (Quiz Interaktif)
- Featured Brands
- Community Reviews
- Personalized Recommendations
- Wishlist Functionality
